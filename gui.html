Feature Selection Widget¶

class ilastikrag.gui.feature_selection_dialog.FeatureSelectionDialog(channel_names, feature_names, default_selections=None, parent=None)[source]¶

A UI for selecting edge features on a per-channel basis.

Example usage:

channel_names = ['Grayscale', 'Membranes']
feature_names = ['standard_edge_mean', 'standard_edge_count',
                 'standard_sp_mean', 'standard_sp_count',
                 'standard_edge_quantiles', 'standard_edge_quantiles_10', 'standard_edge_quantiles_90']

default_selections = { 'Grayscale': ['standard_sp_mean', 'standard_sp_count'],
                       'Membranes': ['standard_edge_quantiles'] }

dlg = FeatureSelectionDialog(channel_names, feature_names, default_selections)
dlg.exec_()
if dlg.exec_() == QDialog.Accepted:
    print dlg.selections()
__init__(channel_names, feature_names, default_selections=None, parent=None)[source]¶
Parameters:
  • channel_names – list of str The user will be shown a separate checklist of feature options for each channel.
  • feature_names – list of str Feature names, exactly as expected by compute_features(). The features will be grouped by category and shown in duplicate checklist widgets for each channel.
  • default_selections – dict, str: list-of-str Mapping from channel_name -> feature_names, indicating which features should be selected by default for each channel.
  • parent – QWidget
selections()[source]¶

Return the user’s choices as a dictionary (channel -> feature names). Call this after the dialog has been accepted.
